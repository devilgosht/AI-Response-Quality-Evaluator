# Milestone 2 Report

## Objective
Implemented the evaluation agents for assessing AI-generated responses.

## Components Developed

- Relevance Judge Agent
- Accuracy Judge Agent
- Hallucination Detection Agent
- Completeness Agent
- Verdict Agent
- Validation Module
- Streamlit Evaluation Interface

## Features

- Calculates relevance score with explanation.
- Calculates factual accuracy with supporting evidence.
- Detects hallucinated or unsupported claims.
- Computes completeness score.
- Generates overall verdict.
- Validates using benchmark question-answer pairs.

## Technologies Used

- Python
- Streamlit
- Sentence Transformers
- FAISS
- Git
